# Security Policy

## Supported Versions

Currently there is no version system as 0.1 is not ready. 

| Version | Supported          |
| ------- | ------------------ |
| Curryvim   | :white_check_mark: |
| Anythin else   | :x:                |

## Reporting a Vulnerability

If you find a Vulnerability, please open an issue describing it. This should include how it works and how one can fix it.
